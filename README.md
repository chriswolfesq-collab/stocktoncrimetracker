# Stockton Crime Tracker V3

A GitHub Pages friendly crime/calls-for-service dashboard for Stockton.

## Files
- `index.html`
- `style.css`
- `app.js`

## How to run locally on Mac

Open Terminal in this folder and run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## What changed in V3
- Defaults to Last 24 Hours
- Date presets:
  - Last 24 Hours
  - Today
  - Yesterday
  - Last 7 Days
  - Last 30 Days
  - This Month
  - Custom Range
- Dashboard counters
- Marker clustering
- Priority marker colors
- Category filter
- Call type filter
- District filter
- Search box
- Auto-refresh option
